本政策适用于所有访问或使用 51API 服务、API、控制台、文档和相关工具的用户。你使用 51API，即表示同意遵守本政策以及适用的服务条款。

## 禁止用途

你不得使用 51API 从事、协助或促成以下行为：

- 违反适用法律法规、监管要求、法院命令或第三方权利。
- 生成、传播或协助实施恶意软件、漏洞利用、凭据窃取、未授权入侵、绕过安全控制或破坏性自动化。
- 生成、传播或协助制作儿童性虐待材料、非自愿性内容、性剥削内容或针对未成年人的不当内容。
- 生成或传播明确鼓励自残、暴力极端主义、恐怖主义、仇恨、骚扰、威胁或人身伤害的内容。
- 冒充他人、伪造身份、进行欺诈、钓鱼、垃圾信息、虚假交易或操纵性活动。
- 未经授权收集、处理、出售或泄露个人信息、敏感信息、凭据、支付信息或商业秘密。
- 规避 51API 的计费、速率限制、风控、审计、访问控制或安全保护。
- 将服务用于高风险决策，而没有适当的人类审核、合规评估和安全控制。

## 受限用途

以下场景需要你自行完成必要的合规评估、权限确认和人工审核：

- 医疗、法律、金融、保险、就业、教育、住房、信贷等可能显著影响个人权益的建议或决策。
- 安全研究、漏洞验证、渗透测试、红队演练或双用途网络安全内容。
- 自动化内容发布、用户画像、风险评分、身份验证或大规模数据处理。
- 面向未成年人、受监管行业或跨境数据传输的产品。

## 用户责任

你应对通过账号、API Key、组织成员或集成系统发起的请求负责。你需要维护账号安全，合理配置权限、限额、日志和审查流程，并确保终端用户了解必要的限制。

如果你发现 API Key 泄露、异常调用、滥用风险或违反本政策的行为，应立即撤销相关凭据，并联系 51API 支持渠道。

## 执行措施

51API 可以根据风险程度采取限流、暂停模型、撤销 Key、限制账号、终止服务、删除违规内容、保留证据或依法配合主管机关等措施。

我们可能会更新本政策。更新后的政策发布后，将适用于后续使用行为。
