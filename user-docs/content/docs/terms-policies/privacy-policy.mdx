本隐私政策说明 51API 在你访问网站、注册账号、使用 API、控制台、文档、支持渠道或相关服务时如何处理个人信息。

## 我们收集的信息

我们可能收集以下信息：

- 账号信息：姓名、昵称、邮箱、手机号、组织名称、登录凭据和认证状态。
- 计费信息：订单、套餐、余额、发票、付款状态、交易标识和必要的账务记录。
- 使用信息：API Key 标识、请求时间、模型、用量、状态码、错误信息、延迟、IP 地址、设备与浏览器信息。
- 支持信息：你通过邮件、工单、聊天、表单或其他渠道提交的问题、附件和沟通记录。
- 内容数据：你通过 API 或控制台提交的提示、输入、上传内容、模型响应和相关元数据，具体保留范围以产品配置、法律要求和双方约定为准。

## 使用目的

我们会为以下目的处理信息：

- 提供、维护、计费、结算和改进 51API 服务。
- 完成账号注册、登录验证、权限管理、客户支持和安全通知。
- 监控服务质量、排查错误、分析用量、执行限额和防止滥用。
- 履行法律法规、合规审计、争议处理、合同义务和监管要求。
- 在你同意或法律允许的情况下，发送产品更新、服务公告或运营信息。

## 共享与委托处理

我们可能与必要的第三方服务提供商共享或委托处理信息，包括云服务、模型供应商、支付服务、邮件短信服务、日志监控、安全风控、客户支持和数据分析服务。

我们会要求服务提供商仅在授权范围内处理信息，并采取合理的安全保护措施。除法律要求、保护权利、执行条款或获得授权外，我们不会出售你的个人信息。

## 保存期限

我们会在实现本政策所述目的所需期间内保存信息。账号、计费、审计、安全、争议处理和合规记录可能在账号关闭后继续保存一段时间。超过必要期限后，我们会删除、匿名化或以法律允许的方式处理相关信息。

## 安全措施

我们采用访问控制、加密传输、密钥隔离、日志审计、权限管理、监控告警和内部流程等措施保护信息安全。没有任何互联网服务可以保证绝对安全；你也应妥善保管账号、密码、API Key 和设备环境。

## 你的权利

在适用法律允许的范围内，你可以请求访问、更正、删除、导出或限制处理你的个人信息，也可以撤回同意、关闭账号或投诉处理方式。为保护账号安全，我们可能需要验证你的身份后再处理请求。

## 跨境传输

如果你在不同国家或地区使用 51API，信息可能会被传输、存储或处理到你所在地区以外的地方。我们会根据适用法律采取必要的合规措施。

## 未成年人

51API 不面向未达到当地法定年龄的未成年人直接提供服务。若你代表组织、学校或家庭场景使用服务，应确保具备必要授权并采取适当保护措施。

## 政策更新

我们可能不时更新本隐私政策。更新后继续使用服务，即表示你了解更新后的处理方式。重大变更会通过合理方式通知。
